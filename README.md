# Projeto de Análise de Vendas

Este projeto foi desenvolvido com o objetivo de praticar e consolidar meu aprendizado
em SQL e EXCEL, e iniciar a construção do meu portfólio na área de dados.

A análise foi realizada com base em uma criação simples de dados de vendas, permitindo
explorar consultas SQL básicas e gerar algumas análises sobre os dados.

---
## Ferramentas utilizadas
- SQL Server
- EXCEL
- Github

---
## Estrutura do projeto

O projeto está organizado nos seguintes arquivos:

# Criacao_Banco_de_Dados_e_Tabela.sql
É o script responsável pela criação do banco de dados e da tabela utilizadas no
projeto.

# Insercao_de_Dados.sql
É o script utilizado para inserir os dados na tabela de vendas.

# Analise_completa.sql
É o arquivo contendo todas as consultas SQL utilizadas para realizar as análises.

# Dados_Vendas.xlsx
É o arquivo contendo os dados utilizados para alimentar a tabela de vendas.

---

## Análises realizadas

Durante o projeto foram realizadas as seguintes análises:

- Vendas por Vendedor
- Vendas por Categoria
- Vendas por Estado
- Vendas por Mês

Essas análises foram realizadas utilizando funções de agregação como:
- SUM
- COUNT
- AVG
- GROUP BY

---

## Objetivo do projeto

O principal objetivo do projeto foi práticar a criação de tabelas em SQL, inserção
de dados, consultas SQL e a organização de um pequeno projeto de análise de dados.

## Aprendizados durante o projeto

Durante o desenvolvimento deste projeto também utilizei ferramentas de Inteligência
Artificial como apoio para esclarecer dúvidas e entender melhor alguns conceitos.
Essa experiência foi importante para aprender a utilizar a IA como ferramenta de estudo
e apoio no desenvolvimento.

Alguns pontos que se destacaram durante o projeto forma:

- Utilzação de comando 'Exec sp_rename' para corrigir o nome de uma coluna na tabela.
- Melhor compreensão da estrutura de criação de tabelas e inserção de dados em SQL.
- Entendimento básico de como dados podem ser organizados entre EXCEL e SQL para análise.

Mesmo sendo um projeto simples, ele representou um avanço importante no meu processo de
aprendizado em análise de dados.
